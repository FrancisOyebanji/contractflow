# Product Roadmap — ContractFlow

Roadmap is outcome-based: each theme is tied to a metric it is expected to move. Sequencing balances customer value, commercial impact, and platform investment.

## Now (Q3 2026) — Prove the core loop

| Initiative | Outcome / metric | Notes |
|---|---|---|
| AI clause review at intake | ≥80% of beta contracts receive ≥1 actionable flag | Rule + LLM hybrid; template library per org |
| Signature orchestration (DocuSign API) | Signature stage time ↓ 40% | Webhook-driven status; signer sequencing |
| Pipeline dashboard + stage SLAs | 100% of stalled contracts surfaced within 24h | Configurable SLA per stage |

## Next (Q4 2026) — Expand the workflow

| Initiative | Outcome / metric | Notes |
|---|---|---|
| Renewal radar | Zero missed renewal/auto-renewal deadlines among beta orgs | Extracts renewal terms at intake; calendar + alerts |
| Adobe Sign + generic webhook integration | Unblocks 30% of pipeline currently on non-DocuSign vendors | Provider-agnostic envelope abstraction |
| Approval routing rules | Review stage time ↓ 25% | Value- and type-based routing (e.g., >$1M → legal) |
| Audit trail & role-based access | Enterprise security review pass | Prereq for mid-market deals |

## Later (H1 2027) — Platform & intelligence

| Initiative | Outcome / metric | Notes |
|---|---|---|
| Portfolio risk benchmarking | Attach rate of analytics add-on ≥ 25% | Cross-portfolio clause benchmarks (anonymized) |
| Public API + integration marketplace | 3 partner-built integrations live | Yardi/MRI-class property management systems first |
| AI-drafted amendment suggestions | Counsel time per contract ↓ 30% | Human-in-the-loop; counsel approves all output |
| EU data residency | Opens EU segment (est. +20% TAM) | Pending discovery with EU prospects |

## Prioritization method

Initiatives are scored quarterly with RICE (Reach, Impact, Confidence, Effort), then sanity-checked against three strategic filters: does it shorten cycle time, does it deepen trust in AI review, does it expand the integratable surface? Items failing all three are cut regardless of score.
